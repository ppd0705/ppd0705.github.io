package main
import "fmt"
func main() {
    var a string = "Runoob"
    fmt.Println(a)

    var b, c int = 1, 2
    fmt.Println(b, c)
    
    var aa int
    fmt.Println(aa)
    
    var bb = true
    // bb := true 
    fmt.Println(bb)
    
}
  • 指定变量类型，如无初始化则为对应类型的默认值。数值型为0，布尔型为false，字符串为""
  • 可以根据值自动判定类型
  • 可以通过:= 申明变量， 但仅能在函数中使用
  • 变量可分为局部变量和全局变量，两种变量名称可以相同，优先考虑局部变量
  • 全局变量一般使用因式分解格式

引用

  • 基本类型属于值类型，变量直接指向内存中的值
  • 其他类型属于引用类型，变量指向值坐在的内存地址

其他

  • 局部变量申明后必须被使用
  • 可以使用a, b = b, a交换同类型的变量
  • 可以用空白标识符_用来抛弃值
